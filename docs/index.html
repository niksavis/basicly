basicly

One command installs a complete coding-agent harness — instructions, skills, and git gates — projected from a single YAML catalog.

Get started GitHub Architecture

Why

Coding agents read guidance from many places — CLAUDE.md, AGENTS.md, copilot-instructions.md, skill folders, hook configs — and keeping them consistent by hand does not scale. basicly keeps one catalog and projects it into every format your agents expect, then guards the result with deterministic gates.

Instructions

Agent instruction files generated from shared fragments — edit one YAML source, every target regenerates consistently.

Skills

Reusable runbooks projected to .claude/skills and .agents/skills, discoverable by any agent that reads the SKILL.md convention.

Gates

Git hooks across pre-commit, commit-msg, and pre-push — plus Claude Code and Copilot agent hooks — that block bad changes whether or not the guidance was followed.

Install

Into any git repo, with uv on the machine: 

$ uvx --from git+https://github.com/niksavis/basicly@v0.2.0 basicly install

No uv or Python yet? The bootstrap shim installs uv first, then runs the same command: 

$ curl -fsSL https://raw.githubusercontent.com/niksavis/basicly/main/.scripts/bootstrap.sh | sh

Windows (PowerShell):

> powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/niksavis/basicly/main/.scripts/bootstrap.ps1 | iex"

Upgrading is the same command with a newer pin — install is an idempotent converge. basicly uninstall removes everything managed; your overlay survives.

How it works

You author guidance once, as versioned YAML sources. The projector renders what each tool natively reads, and a drift gate keeps it honest.

  • Yours stays yours. Overrides and additions live in .basicly-local/; installs and upgrades never touch them.
  • Scoped to your stack. --technologies python,zsh ships only the catalog content that matches.
  • Deterministic first. Everything is plain git-tracked files — no daemon, no hidden state, no network calls at build time.
